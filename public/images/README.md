# ABLE Logo Assets

## Logo File
- **Filename**: `logo.svg`
- **Format**: SVG (Scalable Vector Graphics)
- **Color**: #DE7425 (Orange)

## Logo Design
The ABLE logo features a stylized italic "i" shape that represents:
- **Circle (top)**: Care, attention, and focus on individuals
- **Curved path (body)**: Patient journey, workflow, and progress
- **Italic style**: Movement, dynamism, and forward progress

## Usage Guidelines

### Header Logo
```html
<img src="images/logo.svg" alt="ABLE Logo" class="logo-icon">
```

### Dimensions
- **Desktop**: 35px × 50px
- **Mobile**: 28px × 40px
- **Original**: 100 × 140 viewBox

### Spacing
- Minimum clear space: 10px on all sides
- Always pair with "ABLE" text and tagline

### Color Variations
- **Primary**: #DE7425 (Orange) - Default
- **White**: #FFFFFF - For dark backgrounds (if needed)
- **Dark**: #333333 - For light backgrounds (if needed)

### Don'ts
- ❌ Don't change the proportions
- ❌ Don't rotate the logo
- ❌ Don't add effects (shadows, gradients) to the logo itself
- ❌ Don't use on busy backgrounds
- ❌ Don't change the orange color in standard usage

### Approved Backgrounds
- ✅ White
- ✅ Light gray (#F8F9FA)
- ✅ Warm white (#FFF8F3)
- ✅ Dark backgrounds (with sufficient contrast)
