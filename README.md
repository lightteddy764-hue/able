# ABLE - A Better Life Enabled

A modern, responsive landing page for ABLE healthcare workflow management system with smooth animations and professional design.

## 🎨 Color Scheme

**Orange Core Theme:**
- Primary Color: `#DE7425` (Vibrant Orange) - Represents energy, enthusiasm, and warmth
- Secondary Color: `#F59E52` (Light Orange) - Represents approachability and optimism
- Accent Color: `#C85A11` (Deep Orange) - Represents strength and reliability

This color combination was chosen for its:
- Energetic and positive healthcare appearance
- High visibility and attention-grabbing
- Warm, welcoming feel
- Modern, distinctive aesthetic

## ✨ Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Animations**: 
  - Fade-in effects on scroll
  - Floating cards with parallax
  - Animated statistics counter
  - Pulse animations
  - Hover effects on interactive elements
- **Modern UI Components**:
  - Fixed navigation with ABLE logo and tagline
  - Hero section with orange gradient background
  - Feature cards with SVG icons
  - Benefits section with progress chart
  - Call-to-action section
  - Comprehensive footer
- **Performance Optimized**: Debounced scroll events and efficient animations

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open your browser and navigate to:
```
http://localhost:3000
```

### Development Mode

For auto-restart on file changes:
```bash
npm run dev
```

## 🔧 Customization

### Changing Colors

Edit the CSS variables in `public/styles.css`:

```css
:root {
    --primary-color: #2C5F8D;
    --secondary-color: #4CAF50;
    --accent-color: #00BCD4;
}
```

### Modifying Content

Edit `public/index.html` to change text, images, or structure.

### Adding Features

Add new JavaScript functionality in `public/script.js`.
